My HTML Cheat Sheet

Headings

Headings from `h1` to `h6` identify blocks and sections. Every page must have one `h1`.

Paragraphs

Paragraphs of text go in `p` tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in `ul` tags. Each list item goes in an `li` tag.

Types of lists:

  • Unordered lists (`ul`)
  • Ordered lists (`ol`)

Ordered Lists

Ordered lists are numbered and go in `ol` tags. Each list item goes in an `li` tag.

Polya's Problem Solving Framework

  1. Understand the problem
  2. Come up with a plan
  3. Carry out the plan
  4. Go back and improve your solution

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the `href` attribute. Go to DuckDuckGo to search for more information.

Setting `target="_blank"` will open the link in a new window. See MDN for more details.

Images

Images go in `img` tags. The `src` attribute contains the URL of the image. The `alt` attribute contains a description of the image.

Tables are used to display data in rows and columns. They go in `table` tags. Each row goes in a `tr` tag. Each cell goes in a `td` tag.

Artcle

What is an article?

An article is a self-contained piece of content that can be distributed and re-used independently. Articles can be blog posts, news stories, technical documentation, and more.

footer

The footer tag goes at the end of the page and contains information about the page. It can include contact information, copyright information, and more.

header

The header tag goes at the top of the page and contains information about the page. It can include the page title, navigation links, and more.

nav

The nav tag goes at the top of the page and contains navigation links. It can include links to other pages on the site, links to other sites, and more.
